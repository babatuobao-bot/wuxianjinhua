# 中医经典知识查询系统

> 基于《黄帝内经》与《伤寒论》的中医经典知识可视化查询工具，纯静态网页，无需安装，开箱即用。

***

## 📖 项目简介

本项目收录两套独立的中医经典知识查询系统，面向中医学习者、爱好者及临床参考使用。所有内容均基于原典整理，力求准确严谨，以现代交互方式呈现古典医学智慧。

***

## 🗂️ 收录内容

### 一、五运六气查询系统（`index.html`）

基于《黄帝内经》运气学说，系统介绍五运六气的核心理论。

**主要功能：**

* 五运（木、火、土、金、水）的运行规律与临床意义

* 六气（风、寒、暑、湿、燥、火）的周期变化

* 天干地支与运气推演

* 五运六气与季节、疾病的对应关系

* 古典配色风格，沉浸式阅读体验

**适用人群：** 中医基础理论学习者、运气学说研究者、传统文化爱好者

***

### 二、伤寒论知识查询系统（`shanghan.html`）

基于东汉·张仲景《伤寒论》，构建六经辨证知识库，支持多维度检索。

**主要功能：**

| 模块      | 说明                       |
| ------- | ------------------------ |
| 🏠 首页   | 六经辨证总览，快速导航入口            |
| 🩺 症状查询 | 按症状组合智能匹配证候与方剂           |
| 📋 证候   | 六经分类，涵盖主症、病机、鉴别诊断、原文出处   |
| 💊 方剂   | 组成、用量（汉代换算）、功效、主治、禁忌、煎服法 |
| 🔤 症状   | 各症状在六经中的不同表现与对应方剂        |
| 🔍 全文搜索 | 支持证候、方剂、症状关键词实时检索        |

**收录范围：**

* **六经证候**：太阳、阳明、少阳、太阴、少阴、厥阴全六经

* **核心方剂**：桂枝汤、麻黄汤、白虎汤、大承气汤、小柴胡汤、四逆汤等经典方剂及变方

* **原文出处**：每条证候均标注原典条文编号，便于溯源核查

* **汉代计量换算**：1两=5克，1斤=80克，附注现代用量参考

**适用人群：** 中医学生、临床医师、伤寒论研究者、经方爱好者

## 💻 本地使用

无需安装任何依赖，直接用浏览器打开 HTML 文件即可：

```bash
# 克隆仓库
git clone https://github.com/babatuobao-bot/babatuobao-bot.github.io.git

# 用浏览器打开
open index.html        # 五运六气
open shanghan.html     # 伤寒论
```

## 🛠️ 技术说明

* **纯静态网页**：HTML + CSS + JavaScript，零依赖，零后端

* **无需网络**：所有数据内嵌于页面，离线可用

* **响应式设计**：适配手机、平板、桌面各种尺寸

* **无障碍友好**：语义化标签，支持键盘导航
